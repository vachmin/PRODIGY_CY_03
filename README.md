# 🔐 GUI Password Complexity Checker

A simple Tkinter GUI app to check password strength live as you type.

## Features

✅ Live password strength feedback  
✅ Checks for:
- Minimum 8 characters
- Uppercase letters
- Lowercase letters
- Numbers
- Special characters

## Usage

```bash
python -m password_gui_tool.gui
